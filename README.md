# Dimensionality Reduction: Manual PCA Implementation 📉

### Project Overview
This project focuses on **Principal Component Analysis (PCA)**, implemented from scratch using Singular Value Decomposition (SVD). The goal is to reduce the dimensionality of complex datasets while preserving the maximum amount of variance (information).

Developed as part of the **Master’s in Data Science at the University of Vienna**, this project demonstrates a deep mathematical understanding of linear algebra and its applications in data compression and visualization.

### Key Technical Features
* **Custom PCA Engine:** Implemented via SVD (Singular Value Decomposition) to find orthogonal axes of maximum variance.
* **Explained Variance Analysis:** Calculated and visualized how many principal components are needed to retain specific levels of information.
* **Data Visualization:** Projected high-dimensional data into 2D/3D spaces to identify hidden structures and clusters.
* **Feature Compression:** Demonstrated the ability to transform hundreds of variables into a few meaningful components.

### Tech Stack
* **Language:** Python 3.10+
* **Mathematics:** NumPy (Matrix operations, SVD).
* **Visualization:** Matplotlib, Scipy.
